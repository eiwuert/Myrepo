Image Help

To attach an image to an outgoing email or fax, there are two options:

  • Attach an image to the template.
  • Use an external image located on a webserver.

Attachment
To attach an image to the template, make the image tag's source the name of the attached image. So if the image name is unique_name.jpg, then the image tag should look like: 

<img src="unique_name.jpg">
The image will need to be attached to the template once the template has been saved if the attachment does not already exist (i.e. editing an existing template). Condor will then use the attachment to display the image in outgoing emails and faxes.

External Image
To use an external image, point the image to a valid image URL located on the web. When the fax is created or an email viewed, it will attempt to load the image from the external source. 

<img src="http://www.example.com/image_name.jpg">

The downside to using an external image is that if the image changes, it will change what the user sees and it will not have a historical copy inside of Condor.

Cover Sheet Help

Cover sheets can be created just like any other template. Currently only tokens populated by the eCash application will display anything. You must select Cover Sheet as the Template Type for the template to be created as a cover sheet.

There is one special token when creating a cover sheet. To display the number of pages being sent after the cover sheet, insert the following token:

@@fax_pages@@

This is a special token used by Condor to determine where in the cover page you want the page count to appear.
